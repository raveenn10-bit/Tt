SERENDIB TRAVEL WEBSITE

1. Extract the ZIP file.
2. Open index.html in a browser.
3. Edit text, prices and contact details inside index.html.
4. Upload the entire folder to Netlify, GitHub Pages, cPanel or any static host.

Included:
- index.html
- style.css
- script.js
- local SVG assets

The form is a front-end demo. Connect it to Netlify Forms, Formspree, Firebase or your own backend for real submissions.
